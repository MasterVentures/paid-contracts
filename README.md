# Ignition Crowdsale

Before to Start
---
#### Validate Develop Environment
---
- Node v14.16.0 or +
- Solc v0.8.0 or +
- Hardhat v2.2.1 or +

#### For Testing (Like Ganache for Hardhat)
---
```
npx hardhat node
```
#### Start migration
---
```
npx hardhat run --network localhost --verbose
```

#### Start truffle test
---
```
npx hardhat test
```

## environment variables (.env file)

INFURAKEY=
